Manual Configuration

\n\n
\n

\n Manual configuration involves editing your own WSGI configuration\n file in /var/www/.\n Usually this imports a WSGI-compatible application which\n you\'ve stored elsewhere\n

\n

\n When you click "Next", we will create a WSGI file for you,\n including a simple "Hello World" app which you can use to get\n started, as well as some comments on how to use other frameworks.\n

\n

\n You will also be able to specify a virtualenv to use for your\n app.\n
